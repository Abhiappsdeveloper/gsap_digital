================================================================================
TROUBLESHOOTING QUICK FIXES:
================================================================================

ISSUE: "Port 8000 already in use"
FIX: Edit server.js, change PORT = 8000 to 8001 or 3000

ISSUE: "Cannot GET /"
FIX: Make sure you're in correct folder:
     cd C:\Users\NS\Documents\ABHIJEET\git\gsap_digital

ISSUE: "Images not loading"
FIX: Check digital_files/ folder exists with all WebP files

ISSUE: "3D canvas blank/black"
FIX: Wait 1-2 seconds for Three.js to load
     Check console (F12) for errors
     Make sure browser supports WebGL

ISSUE: "Menu won't open"
FIX: Check console (F12) for JavaScript errors
     Verify hamburger menu button is clickable

ISSUE: "No animations"
FIX: Check console for GSAP loading errors
     Try refreshing page (Ctrl+R)
